# Coodra Premium Landing (Local Prototype)

This folder contains a local-only scaffold for a premium Coodra landing page.

## Included

- `index.html`: semantic landing structure with sections for hero, how-it-works, decision card, trust rail, CTA, and footer placeholders.
- Animation hooks via data attributes: `data-reveal`, `data-stagger`, `data-countup`, `data-marquee`, `data-step-node`.
- Linked local assets: `styles.css` and `app.js`.

## Notes

- Prototype only; no Shopify theme files were modified.
- Copy is placeholder-ready and brand-consistent for future content and motion polish.
