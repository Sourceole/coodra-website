## 2026-04-24 — Integrations connect scaffolding (dashboard)

### What was implemented
- Dashboard integration buttons now call backend start endpoint instead of hardcoded Shopify-only redirects.
- Frontend wiring:
  - `public/wh-command-center.web.js`
    - Added `INTEGRATIONS_START_API` (`/api/integrations/start`).
    - Added `requestProviderLaunch(provider, shopDomain)`.
    - `startPerformanceConnect` now:

      - Opens returned launch URL in popup/new tab.

  - `src/pages/Dashboard.tsx`
    - Asset version bumped to: `2026-04-24-integrations-oauth-scaffold-67`.

- Backend wiring (without adding extra serverless function count):
  - `sourceole-ai-backend/api/performance.js`
    - Added integration state signing/verification (HMAC).
    - Added provider config resolver for Shopify, Square, Clover, Lightspeed, Moneris.
    - Added OAuth authorize URL builder and token exchange helper.
    - Added token encryption helper for stored provider tokens.
    - Added new views:
      - `view=integration_start` (POST, auth required)
      - `view=integration_callback` (GET, state-verified callback)

  - `sourceole-ai-backend/vercel.json`
    - Added rewrites:
      - `/api/integrations/start` → `/api/performance?view=integration_start`

### Notes
