## Quick Command Reference

For coodra.com, when analyzing SEO:
1. Fetch GSC non-branded keywords — what are the actual opportunities?
2. Run content quality check — do pages have E-E-A-T signals?
3. Audit schema — is it validating AND earning rich snippets?
4. Check rendering — is Googlebot seeing the same content as users?
5. Internal linking — are pages linking to each other logically?
